Common Phrases

Greetings

  • English: Hello / Good morning / Good evening
  • Spanish: Hola / Buenos días / Buenas noches
  • Mandarin: 你好 (Nǐ hǎo) / 早上好 (Zǎoshang hǎo) / 晚上好 (Wǎnshàng hǎo)
  • Japanese: こんにちは (Konnichiwa) / おはようございます (Ohayou gozaimasu) / こんばんは (Konbanwa)
  • Filipino: Kumusta / Magandang umaga / Magandang gabi

Polite Phrases

  • English: Thank you / Excuse me / Please
  • Spanish: Gracias / Perdón / Por favor
  • Mandarin: 谢谢 (Xièxiè) / 对不起 (Duìbuqǐ) / 请 (Qǐng)
  • Japanese: ありがとうございます (Arigatou gozaimasu) / すみません (Sumimasen) / お願いします (Onegaishimasu)
  • Filipino: Salamat / Paumanhin / Pakiusap

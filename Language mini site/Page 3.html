Lesson 3 — Numbers & Time

In this lesson, you’ll learn how to count from 1–5 and tell simple time expressions in different languages.

Counting 1–5

1, 2, 3, 4, 5uno, dos, tres, cuatro, cinco (Spanish)
1, 2, 3, 4, 5un, deux, trois, quatre, cinq (French)
1, 2, 3, 4, 5ichi, ni, san, yon, go (Japanese)
1, 2, 3, 4, 5isa, dalawa, tatlo, apat, lima (Filipino)
1, 2, 3, 4, 5uno, due, tre, quattro, cinque (Italian)

Telling Time

What time is it?¿Qué hora es? (Spanish)
It’s 3 o’clockIl est trois heures (French)
It’s 7 o’clockShichi-ji desu (Japanese)
It’s 12 o’clockAlas dose (Filipino)
It’s 9 o’clockSono le nove (Italian)

Practice Activity

1. Count 1–5 in Japanese → ichi, ni, san, yon, go

2. Say “It’s 3 o’clock” in French → Il est trois heures

3. Say “What time is it?” in Spanish → ¿Qué hora es?
